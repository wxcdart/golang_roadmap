# regexp examples

This folder contains examples demonstrating Go's `regexp` package:

- `regexp.Compile` vs `regexp.MustCompile`
- `MatchString`, `FindString`, `FindAllString`, `FindStringSubmatch`
- Named capture groups and `SubexpNames()`
- `ReplaceAllString` and `ReplaceAllStringFunc`
- `Split` and `QuoteMeta`
- Performance notes and common pitfalls

Run:

```bash
cd golang_roadmap/03_std_lib/07_regex
go run regex_examples.go
```
