# Testing

This folder demonstrates basic testing with the Go `testing` package.

Contents:

- `mathutil.go` — small package with a couple of functions.
- `mathutil_test.go` — table-driven tests demonstrating `t.Run` and subtests.

Commands:

```bash
cd golang_roadmap/04_Tooling_testing_and_code_quality/01_testing
go test -v
go test -cover
```
